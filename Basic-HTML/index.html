The Basic Tags of an HTML Document

Heading Tags

Heading tags are denoted by "h1" to "h6", that varies in font sizes

Paragraph tags

A paragraph tag is denoted by "p". It encloses a block of text and tells the computer it's a paragraph

In-Text Style CSS Tags

Tags such as "strong" and "em" are used to bold and Italisize a block of text.

Bullet Tags
